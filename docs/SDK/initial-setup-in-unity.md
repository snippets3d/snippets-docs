# 4. Initial setup in Unity

## 4.1 Web Portal preparation

Create/publish snippet sets in the web portal and ensure they are downloadable by pressing the
“Publish” button. You must not leave the tab while the Snippet is publishing for the process to finalize
successfully.

![Screenshot](/assets/images/sdk-documentation/page-07-image-01.png)

<p class="screenshot-caption">Press publish to make snippet set accessible to SDK</p>

---

## 4.2 Log in

1. `Snippets > Log In`
2. Enter credentials (same as on the web portal)
3. Confirm successful login

![Screenshot](/assets/images/sdk-documentation/page-08-image-01.png)

<p class="screenshot-caption">Log into the Snippets SDK</p>

---

## 4.3 Open set management

1. `Snippets > Import or Update Snippet `Sets`
2. Review set statuses and actions

**Status labels:**

- [Imported]
- [Local Deprecated]
- [Processing/Unpublished]
- [Update Processing]

![Screenshot](/assets/images/sdk-documentation/page-08-image-02.png)

<p class="screenshot-caption">Snippet Sets Window - for importing, updating, removing or viewing your snippet sets</p>
