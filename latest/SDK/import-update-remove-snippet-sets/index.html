    6. Import, update, remove snippet sets

    6.1 Import

    1. Click Import
    2. Select template prefab
    3. Confirm and wait for completion

    Process:

    • Download raw archive
    • Decompress assets
    • Generate prefabs from template

    Import button from Snippet Sets Window

    Importing Snippets Window

    6.2 Update

    1. Click Update (available only after new changes are published on the web portal on the same set)
    2. Confirm operation Matching prefabs are updated in place, removed snippets are deleted, and new snippets are generated, helping preserve references.

    Update button from Snippet Sets Window

    6.3 Remove

    1. Click Remove
    2. Confirm Removes local generated and raw files for that set.

    Remove button from Snippet Sets Window
